
# AstraCLI AI 🚀

> AI-Powered Desktop Command Line Assistant for Windows  
> Launch apps, search files, automate tasks, monitor system stats, and control your PC using natural language.

![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-success)
![License](https://img.shields.io/badge/License-MIT-green)
![PyPI](https://img.shields.io/pypi/v/astracli-ai)
![Downloads](https://img.shields.io/pypi/dm/astracli-ai)

---

## ✨ Features

✅ Natural Language Commands  
✅ Open Apps (Chrome, VSCode, Notepad, Calculator)  
✅ Smart File Search Across Drives  
✅ Google Search Instantly  
✅ YouTube Play Commands  
✅ Battery / CPU / Memory Monitoring  
✅ Screenshot Capture  
✅ Shutdown / Cancel Shutdown  
✅ Rich Terminal UI  
✅ Extensible Python Architecture  

---

## 📦 Installation

Install directly from PyPI:

```bash
pip install astracli-ai
````

---

## 🚀 Run AstraCLI

```bash
astra
```
